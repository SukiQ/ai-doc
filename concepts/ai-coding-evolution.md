# AI 编程演进

```
代码补全(2010s) → 智能补全(2021) → 对话式编程(2023) → Vibe Coding(2025)
规则驱动            AI 补全           AI 对话            AI 代理
IDE 功能            编辑器插件         网页对话            终端/编辑器
单行建议            多行建议           代码块生成          项目级操作
```

## 关键里程碑

| 时间 | 事件 |
|------|------|
| 2021 | GitHub Copilot — AI 补全进入实用 |
| 2022 | ChatGPT — 对话式 AI 编程 |
| 2023 | GPT-4 / Claude — 代码理解大幅提升 |
| 2024 | Cursor / Claude 3.5 — AI 原生编辑器 + 长上下文 |
| 2025 | Claude Code — 终端 AI 代理，Vibe Coding 概念提出 |

## 趋势

- 代理能力增强：单文件 → 多模块协调 → 架构设计
- 工具深度集成：数据库/云服务/CI/CD 全链路
- 门槛降低：自然语言成为最重要的"编程语言"
