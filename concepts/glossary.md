# 术语速查

| 术语 | 说明 |
|------|------|
| Agent | 自主规划和执行任务的 AI |
| Chain of Thought | 让模型逐步推理 |
| Context Window | 模型一次处理的最大 token 数 |
| Embedding | 文本→数值向量 |
| Few-shot | 少量示例指导模型 |
| Fine-tuning | 特定数据继续训练 |
| Function Calling | 模型调用预定义函数 |
| Grounding | 输出基于真实依据 |
| Hallucination | 编造看似正确的内容 |
| Hook | 事件触发的回调 |
| MCP | 连接 AI 与外部工具的协议 |
| Prompt | 发送给 AI 的指令 |
| RAG | 结合外部知识检索的生成 |
| Sandbox | 隔离的安全执行环境 |
| Skill | Claude Code 可复用工作流 |
| Temperature | 控制生成随机性 |
| Token | 模型处理文本的基本单位 |
| Tool Use | AI 调用外部工具 |
| Transformer | LLM 基础架构 |
| Vibe Coding | 自然语言描述意图让 AI 生成代码 |
| Zero-shot | 无示例直接完成任务 |
