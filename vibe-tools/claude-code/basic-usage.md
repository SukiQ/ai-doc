# 命令速查

## 斜杠命令

| 命令 | 用途 |
|------|------|
| `/compact [指令]` | 压缩上下文，可指定保留重点 |
| `/cost` | Token 用量 |
| `/review` | Code Review |
| `/init` | 生成 CLAUDE.md |
| `/memory` | 编辑记忆文件 |
| `/model` | 切换模型 |
| `/clear` | 清空对话 |
| `/doctor` | 环境诊断 |
| `/permissions` | 权限管理 |
| `/vim` | Vim 模式 |

## 工具调用标识

| 标识 | 操作 |
|------|------|
| 📖 Reading | 读取文件 |
| ✏️ Edit | 编辑文件（需确认） |
| 🔧 Bash | 执行命令（需确认） |
| 🔍 Searching | 搜索代码 |

## 键盘

`Enter` 提交 | `Shift+Enter` 换行 | `Ctrl+C` 中断 | `Ctrl+D` 退出
