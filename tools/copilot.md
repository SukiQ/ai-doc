# GitHub Copilot

编辑器内 AI 补全 + Chat，补全体验最佳。

## Chat 命令

| 命令 | 功能 |
|------|------|
| `/explain` | 解释代码 |
| `/tests` | 生成测试 |
| `/fix` | 修复问题 |
| `/doc` | 生成文档 |
| `/simplify` | 简化代码 |

## 价格

Free（2000 补全/月，50 对话）| Individual $10/月 | Business $19/月/人

## 技巧

- 写清晰注释触发高质量补全
- Copilot 做补全，Claude Code 做复杂任务
