# Cursor

基于 VS Code 的 AI 原生编辑器，保留全部 VS Code 插件生态。

## 核心功能速查

| 功能 | 快捷键 | 用途 |
|------|--------|------|
| Composer | `Ctrl/Cmd+I` | 多文件 AI 编辑 |
| Chat | `Ctrl/Cmd+L` | 项目级 AI 对话 |
| Tab 补全 | `Tab` | 多行代码预测 |
| Inline Edit | 选中→`Ctrl/Cmd+K` | 选中代码 AI 改写 |

## Chat 引用语法

`@file` 引用文件 | `@web` 搜索 | `@docs` 文档 | `@git` 历史
