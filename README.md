# Vibe Coding 中文指南

面向开发者的 AI 编程实战文档站，基于 [VitePress](https://vitepress.dev) 构建。

**在线访问**：[sukiq.github.io/ai-doc](https://sukiq.github.io/ai-doc/)

## 内容

| 板块 | 说明 |
|------|------|
| 概念 | Vibe Coding、AI 编程演进、LLM 基础 |
| 工具 | Claude Code、Cursor、Copilot、Windsurf 等横向对比 |
| 实践 | 提示工程、新项目搭建、功能开发、重构迁移 |
| 框架 | OpenAI API、Anthropic API、LangChain、LlamaIndex |
| 案例 | 全栈应用、REST API、数据分析、自动化、AI Agent |

## 本地开发

```bash
npm install
npm run dev
```

## 部署

推送到 `main` 分支自动通过 GitHub Actions 部署至 GitHub Pages。
