# Vibe Coding 中文文档网站

VitePress 文档站，面向资深开发者的 Vibe Coding 速查指南。

## 技术栈

VitePress ^1.6 + Vue 3 | 品牌色 `#D97757`（Claude Code 暖橙）| Lucide 扁平线框图标

## 结构

```
├── concepts/                    # Vibe Coding 概念（5 页）
├── tools/                       # AI 编程工具对比（6 页）
├── vibe-tools/claude-code/      # Claude Code 速查（10 页）
├── practices/                   # 实践技巧（5 页）
├── frameworks/                  # API/框架速查（4 页）
└── cases/                       # 实战案例（5 页）
```

共 6 板块 35 页 + 首页。导航：概念（直链）+ 更多（下拉菜单）。

## 运行

```bash
npm run dev      # 开发
npm run build    # 构建
npm run preview  # 预览
```

## 文档风格

面向资深程序员：无安装教程、无基础概念解释，只保留速查表、关键配置、核心代码片段和实战要点。
